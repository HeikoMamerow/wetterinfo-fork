== Description ==

Fork of the Wetter.info Wetter Gadget. The origin plugin (https://wordpress.org/plugins/wetterinfo-wetter/) gives you mixed contend: path with http. This fork has a tweak for this problem.

Aktuell nur in deutscher Sprache verfügbar.

== Installation ==

1. Plugin als Zip oder Ordner in das /wp-content/plugins/` Verzeichnis hochladen
2. Plugin aktivieren
3. Wetter.info Konfiguration aufrufen und Ort auswählen. 
4. Design erstellen
5. Widget an die richtige Position setzen und fertig.
